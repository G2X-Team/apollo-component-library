# TextInput

TextInput is a foundational and core component that allows users to enter text by giving them a
text input field. It comes with very usefull props to customize not only the look & feel but
functionality as well. It is a great component to use in forms, search bars, and other places
where users need to enter text.

### Disabled TextInput

With the use of `disabled` prop you can disable the TextInput component. This comes in handy
when you want to disable a TextInput component till the user performs a specific action.

<Canvas>
    <Story name="Disabled TextInput" args={{ label: 'Disabled TextInput', disabled: true }}>
        {DisabledTextInput.bind({})}
    </Story>
</Canvas>

### Password TextInput

With the use of `password` prop you can make the TextInput component a password field. This is usefull
when you want to hide the text entered by the user.

<Canvas>
    <Story name="Password TextInput" args={{ label: 'Password', password: true }}>
        {PasswordTextInput.bind({})}
    </Story>
</Canvas>

### Required TextInput

With the use of `required` prop you can make the TextInput component required. This is usefull
when you want to make sure the user enters some text before submitting the form.
